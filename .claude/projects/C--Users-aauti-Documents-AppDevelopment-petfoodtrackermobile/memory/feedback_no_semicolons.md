Do not use semicolons at the end of statements in `.js`, `.ts`, `.jsx`, or `.tsx` files.

**Why:** User explicitly requested this style.

**How to apply:** All new and modified JS/TS/JSX/TSX code in this project. Applies everywhere — imports, variable declarations, expressions, etc.
