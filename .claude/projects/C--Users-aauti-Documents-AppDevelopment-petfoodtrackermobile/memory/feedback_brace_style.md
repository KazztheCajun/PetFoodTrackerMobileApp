Use Allman-style brace placement: open `{` on its own new line for function bodies, if/else, and other control flow blocks.

**Why:** User explicitly requested this style.

**How to apply:** Any new or modified TypeScript/TSX code in this project. Does not apply to JSX attribute expressions or inline object literals.
